لو عندي referrals، الخصم بيطبق على السنوي كمان؟

آه — الخصم بيطبق على كلا الاشتراكين. كل ما زاد عدد الـ referrals كل ما زاد الخصم.

الكود بيشتغل على أكتر من portfolio؟

الكود بيفعّل حسابك كله — مش portfolio بعينه. تقدر تعمل edit وتغيير themes على أي portfolio في حسابك طول مدة الاشتراك.

Portfolio

لو انتهى اشتراكي، الـ portfolio بتاعي هيختفي؟

لأ — الـ portfolio بيفضل موجود ومنشور بنفس الـ theme حتى بعد انتهاء الاشتراك. بس مش هتقدر تعمل أي تعديل عليه — زي إضافة مشروع جديد أو تغيير الـ theme — إلا لو جددت اشتراكك.

إزاي أضيف صور (screenshots) لمشاريعي؟

في الـ edit page، كل مشروع عنده حقل لرابط الصورة — بتضيف فيه رابط مباشر للصورة.

أسهل طريقة — GitHub repo:
1. ارفع الـ screenshot جوه الريبو في مجلد assets
2. افتح الصورة على GitHub وانقر Raw
3. انسخ الرابط — هيبدأ بـ raw.githubusercontent.com
4. الصق الرابط في الحقل ✓

تقدر تضيف حتى 4 صور لكل مشروع.

LinkedIn Presence

إيه هي خدمة LinkedIn Presence؟

هي أداة بتحلل GitHub profile بتاعك وبتولد محتوى LinkedIn جاهز للنشر — Score لحسابك، اقتراحات تحسين، headline، about section، وpost لكل مشروع. كل ده بالعربي أو الإنجليزي حسب اختيارك.

الـ posts اللي بتتولد، مين بينشرها؟

أنت بتنشرها بنفسك — الأداة بتولد المحتوى جاهز وأنت بتنسخه وتنشره على LinkedIn. مفيش ربط مباشر بحسابك على LinkedIn.

الـ Score بيتحسب إزاي؟

الـ Score من 100 بيتحسب بناءً على: جودة الـ README، انتظام الـ commits، وجود descriptions للـ repos، تنوع المشاريع، واكتمال الـ profile. كل عنصر عنده وزن مختلف في الحسبة النهائية.

أقدر أولّد المحتوى أكتر من مرة؟

آه — تقدر تولّد في أي وقت. كل مرة بتضيف مشروع جديد على GitHub أو بتحسّن حسابك، تقدر ترجع وتولّد من تاني وتاخد محتوى محدّث.
